## 1. 프로젝트 개요

### 1.1 프로젝트 이름
- 서비스명: **Campaign Management System**
- 한글명: **캠페인 관리 시스템**

### 1.2 목적
여러 플랫폼(공모전 사이트, 커뮤니티, SNS 등)에 캠페인을 게재하고, 그 성과를 통합적으로 추적 및 관리하는 시스템

### 1.3 주요 기능
- **채널 관리**: 공모전 플랫폼, 커뮤니티, SNS 등 게재 채널 정보 관리
- **캠페인 관리**: 캠페인 생성, 일정 관리, 목표 설정
- **게재 추적**: 각 채널별 게재 현황 및 URL 관리
- **성과 분석**: 조회수, 클릭수, 등록수 등 성과 지표 추적
- **통합 대시보드**: 전체 캠페인 성과를 한눈에 확인

---

## 2. 핵심 기능

### 2.1 채널 관리
- **채널 등록**: 이름, URL, 카테고리, 회원수, 일평균 조회수 등록
- **카테고리 분류**: 공모전, 커뮤니티, SNS, 이벤트
- **채널 상태 관리**: 활성/비활성 상태 관리
- **채널 정보**: 담당자, 게재 요구사항 등 관리

### 2.2 캠페인 관리
- **캠페인 생성**: 이름, 설명, 기간, 목표 설정
- **카테고리 관리**: 공모전, 세미나, 이벤트 등
- **예산 관리**: 캠페인별 예산 설정
- **상태 관리**: 계획중, 진행중, 완료, 취소

### 2.3 게재 현황 추적
- **게재 기록**: 캠페인별, 채널별 게재 URL 및 날짜 기록
- **성과 추적**: 조회수, 클릭수, 등록수 실시간 업데이트
- **전환율 계산**: 자동 전환율 계산
- **상태 관리**: 대기중, 게재중, 삭제됨, 만료됨

### 2.4 성과 분석
- **통합 대시보드**: 전체 캠페인 성과 요약
- **일별 트렌드**: 성과 지표의 일별 변화 추이
- **채널별 분석**: 채널별 효과성 분석
- **캠페인 비교**: 캠페인간 성과 비교

### 2.5 데이터 관리
- **내보내기**: JSON 형식으로 데이터 내보내기
- **가져오기**: 엑셀, CSV 파일에서 데이터 가져오기
- **백업**: 정기적인 데이터 백업

## 3. 사용자 스토리

### 3.1 마케팅 담당자
- 새로운 캠페인을 생성하고 목표를 설정한다
- 여러 채널에 캠페인을 게재하고 URL을 기록한다
- 실시간으로 성과를 모니터링한다
- 채널별 효과를 분석하여 최적의 채널을 선정한다

### 3.2 팀 리더
- 전체 캠페인의 진행 상황을 파악한다
- 목표 대비 달성률을 확인한다
- 예산 대비 ROI를 분석한다
- 보고서용 데이터를 추출한다

---

## 4. 데이터베이스 구조

### 4.1 주요 테이블
- **users**: 사용자 정보
- **campaign_categories**: 캠페인 카테고리
- **campaigns**: 캠페인 정보
- **campaign_channels**: 게재 채널 정보
- **campaign_posts**: 게재 기록 및 성과

### 4.2 관계
- 캠페인 → 여러 게재 기록
- 채널 → 여러 게재 기록
- 사용자 → 여러 캠페인
